Title   : Pascal ReadMe

This directory contains the sources to build the 
Pascal Triangle example.

The Pascal example is a simple program which shows 
how semi-hosting works with AEB-1 board. Pascal's
Triangle is calculated on the AEB-1 and the result
are displayed on the host PC.

Sources      Type    Description
===================================================
pascal       .c      contain the program entry 
                     point.
led          .h      MACRO's to switch LED's on and
                     off
Pascal       .apj    Pascal Project File used by
                     APM
